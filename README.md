# 🎓 GradeHub - Exam Grading System

A full-stack web application for managing student exams, grades, and report cards.

## 🛠️ Built With
- **Backend:** Django + Django REST Framework
- **Frontend:** React + Vite
- **Database:** SQLite (development) / PostgreSQL (production)
- **Authentication:** JWT (JSON Web Tokens)

## ✅ Features
- User authentication with roles (Admin, Teacher, Parent)
- Institution and class management
- Student management
- Exam creation and management
- Mark entry with automatic grade calculation
- Student rankings
- Audit logging for mark changes
- PDF Report card generation ->done
- Parent portal (coming soon)->phase 6
- Analytics dashboard (coming soon)->phase 7
